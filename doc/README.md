# naia-relay Documentation

This folder contains the Phase 12 documentation set for `naia-relay`.

- `developer-guide.md` — relay roles, local topologies, runtime architecture, and test layout
- `operator-guide.md` — installation, config sources, transports, examples, and local usage
- `unsupported-v1.md` — v1 unsupported features, current caveats, and expected errors
- `tep-protocol.md` — full Tool Executor Protocol reference
- `rlp-protocol.md` — full Relay Link Protocol reference
- `readiness-file.md` — control-plane readiness file format and usage

`SPEC.md` remains the source of truth for behavior and scope.
